# Digital Design
## Introduction
This repository contains various digital design projects implemented using Hardware Description Languages (HDL), especifically VHDL. The projects cover a range of topics including combinational and sequential logic design, finite state machines, and more complex digital systems.
## Directory Structure
It is recomended to follow the directory structure below for better organization of VHDL projects:  
Project Name/
  ├── HDL/
  │   └── [VHDL source files]
  ├── Modelsim/
  │   └── [Modelsim project files]
  ├── Quartus/
  │   └── [Quartus project files]
  └── Test/
      └── [Testbenches and simulation files]
